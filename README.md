# Application allows keep cooments on klient and server side.
---
## On front-end side was used :
1. HTML5/CSS3
2. Angular 6
3. Bootstrap
4. API requests : "POST"(done), "GET"(done), "DELETE"(not-complited) 
---
### On back-end side was used :
1. Node.js
2. Express. js
3. MongoDB
4. Allow-Control-Allow-Origin(CORS) for reading requests from localhosts...
